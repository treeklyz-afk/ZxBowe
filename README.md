# ZxBowe

React Native App Template with a Premium Dark Gradient Theme.

## ✨ Features
- **Pure Dark Gradient Design**: Custom background with smooth transitions.
- **Modern UI**: Clean and minimal layout.
- **Ready to Build**: Complete structure for React Native development.

## 🚀 Getting Started

### Prerequisites
- Node.js (>= 16)
- React Native CLI
- Android Studio / Xcode

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/treeklyz-afk/ZxBowe.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   - **Android**: `npx react-native run-android`
   - **iOS**: `npx react-native run-ios`

## 🎨 Theme Details
- **Primary Colors**: Dark Navy, Deep Purple, Slate Black
- **Typography**: High-contrast white text with subtle shadows.
